Sobre

DroidFish é uma rica interface gráfica para xadrez , combinada com o potente software de xadrez StockFish.

Características

  • Livro de aberturas
  • Jogar/Voltar ilimitado
  • Copiar/colar de/para área de transferência
  • Modo de análise
  • Modo para dois jogadores
  • Modo computador vs computador
  • Definir posições
  • Amplo suporte a PGN
  • Força de jogo ajustável
  • Softwares UCI de terceiros
  • Tabelas de finais de Gaviota

Dicas

  • Pressione longamente os botões de modo/esquerda/direita para comandos adicionais.
  • Quando estiver jogando novamente uma prtida, altere o modo de jogo para Editar/rejogar partida para poder desfazer/refazer lances.

Livros de aberturas

DroidFish vem com um pequeno livro de aberturas. Se você quiser usar um livro maior, você pode configurar o DroidFish para usar livros externos Polyglot ou em formato CTG:

  1. Copie um ou mais arquivos de livro Polyglot para o diretório DroidFish no cartão SD. Livros Polyglot devem ter a extensão de arquivo .bin. Nota! O sistema de arquivos do Android diferencia maiúsculas de minúsculas, portanto, a extensão deve ser .bin, e não .Bin ou .BIN.
  2. Copie um ou mais arquivos CTG para diretório DroidFish. Um livro CTG é composto por três arquivos com extensões CTG, CTB e .cto. Você deve copiar todos os 3 arquivos.
  3. No menu do programa, ative a função Selecionar livro de aberturas.
  4. Selecione o arquivo de aberturas que você deseja usar.

Arquivos PGN

DroidFish pode carregar partidas de arquivos PGN. Para usar este recurso:

  1. Copie um ou mais arquivos PGN para o diretório DroidFish/pgn no cartão SD.
  2. Pressione longamente o tabuleiro e selecione Carregar partida de arquivo PGN.
  3. Selecione o arquivo e depois a partida dentro do arquivo que você deseja carregar.

Para editar os cabeçalhos, como nomes de jogador, pressione longamente a lista de lances e selecione Editar cabeçalhos. Para editar comentários de um lance, pressione longamente a lista de lances e selecione Editar comentários.

Para salvar a edição de sua partida, pressione longamente o tabuleiro e selecione Salvar partida em arquivo PGN. Selecione o arquivo para salvar e em seguida selecione a posição no arquivo onde será salva a nova partida.

Nota! Se o OI File Manager está instalado, qualquer diretório pode ser usado para carregar/salvar partidas.

Para adicionar a análise do software para a árvore da partida, pressione longamente as informações de análise quando o computador estiver analisando e selecione Adicionar análise. A variante principal atual é adicionada como uma variante na árvore da partida.

No modo de análise, é possível adicionar lances nulos para a árvore da partida. Esteja ciente no entanto que lances nulos não são suportados pelo padrão PGN e podem não funcionar em outros programas de leitura de arquivos PGN.

Arquivos Scid

Se você tem Scid on the go instalado, você pode ler arquivos de banco de dados do Scid diretamente a partir do DroidFish. Pressione longamente o tabuleiro e selecione Carregar partidas de arquivo Scid.

Softwares UCI

DroidFish pode usar softwares UCI de terceiros. Para usar este recurso:

  1. Copie um ou mais softwares UCI para o diretório DroidFish/uci no cartão SD. Nota! Os binários devem ser compilados para o Android.
  2. No menu do programa, escolha Gerenciar softwares de Xadrez e Selecione o software de Xadrez e selecione o mecanismo desejado.

Softwares UCI em servidores remotos

DroidFish pode usar softwares UCI rodando em um servidor remoto. Para usar este recurso:

  1. Instale software de servidor de rede de Xadrez no computador remoto.
    1. Para windows, instale o software de servidor a partir da página Utilitário de Xadrez do computador.
    2. Para Linux, pode ser usado o mini-inetd do pacote tcputils.
  2. Seleccione Gerenciar softwares de Xadrez no menu do programa, criar um novo mecanismo de rede e digite o nome do servidor e número do porto para o mecanismo remoto.
  3. No menu do programa, escolha Gerenciar softwares de Xadrez e Selecione o softwares de Xadrez para selecionar o mecanismo desejado.

Tabelas de finais Gaviota

Para usar Tabelas de finas Gaviota:

  1. Copie arquivos .gtb.cp4 para o diretório DroidFish/gtb no cartão SD.
  2. Use a tela de preferências de Tabelas de Finais para controlar como as tabelas são usadas.
